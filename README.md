# PortalABA

O Portal ABA é um Sistema para profissionais que realizam atendimentos a pessoas do espectro autista. O objetivo do Portal ABA é integrar todos os atendimentos realizados pelos profissionais que acompanham os autistas e aplicam a metodologia ABA, para assim obter um melhor gerenciamento dos processos de cada profissional, e consequentemente, um melhor resultado nos atendimentos realizados. 

A missão do Portal ABA é ajudar na progressão das pessoas com transtorno do Espectro Autistas, proporcionando uma visualização ampla do que é realizado pelos profissionais ABA de forma digital, intuitiva e segura.
